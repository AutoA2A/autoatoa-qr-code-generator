# 二维码生成器

```markdown
# 二维码生成器项目说明

## 网站简介

本项目是一个由多个AI协作完成的二维码生成器网站。该网站提供了一个用户友好的界面，使用户能够轻松生成各种类型的二维码，包括文本、URL、Wi-Fi信息、电子邮件、联系人信息、SMS、电话号码和地理位置信息。网站采用响应式设计，确保在不同设备上都能提供良好的用户体验。

## 页面与功能

本网站共包含8个页面，每个页面都提供不同的功能：

1. **主页**：网站的入口页面，提供对各种二维码类型的概述和快速访问链接。
2. **文本二维码**：允许用户输入文本内容，生成包含该文本的二维码。
3. **URL二维码**：用户可以输入URL，生成指向该URL的二维码。
4. **Wi-Fi二维码**：用户可以输入Wi-Fi的SSID和密码，生成连接到该Wi-Fi的二维码。
5. **电子邮件二维码**：用户可以输入收件人地址、主题和正文，生成发送电子邮件的二维码。
6. **联系人信息二维码**：用户可以输入联系人的姓名、电话号码和电子邮件地址，生成包含该联系人信息的二维码。
7. **SMS二维码**：用户可以输入接收者的电话号码和短信内容，生成发送短信的二维码。
8. **电话号码二维码**：用户可以输入电话号码，生成拨打该电话号码的二维码。
9. **地理位置信息二维码**：用户可以输入地理位置的经纬度或地址，生成指向该地理位置的二维码。

每个页面都包含一个表单，用户可以在其中输入所需的信息，然后点击“生成二维码”按钮。生成的二维码将显示在页面上，用户可以下载或分享该二维码。

## 多 AI 协作与验收

本项目是由多个AI协作完成的。每个AI负责不同的任务，例如设计用户界面、编写前端代码、生成二维码等。AI之间通过GitHub仓库进行协作，每个AI都可以提交代码更改，并与其他AI协作完成项目。

在项目完成后，我们进行了验收测试，以确保网站的功能和性能符合要求。我们测试了每个页面的表单输入和二维码生成功能，确保它们能够正确工作。我们还测试了网站在不同设备和浏览器上的兼容性，确保用户能够在各种设备上获得良好的体验。
```
